## Prerequisites

1. **Node.js** (version 18.0 or higher)

1. Download from: [https://nodejs.org/](https://nodejs.org/)
2. This includes npm automatically



2. **Git** (to clone from GitHub)

1. Download from: [https://git-scm.com/](https://git-scm.com/)


## Installation Steps

# 1. Clone the repository
git clone https://github.com/yourusername/electrozone.git

# 2. Go to project folder
cd electrozone

# 3. Install dependencies
npm install

# 4. Run the project
npm run dev
